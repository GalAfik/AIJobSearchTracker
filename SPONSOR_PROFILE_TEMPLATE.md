## 👋 Hi, I'm Gal Afik!

I'm a software developer passionate about creating tools that help job seekers succeed. My main project is **Job Search Tracker** - a comprehensive desktop application that helps people organize and manage their job search journey.

### 🎯 What I'm Working On

**Job Search Tracker** is a free, open-source WPF application that helps job seekers:
- 📝 Track job applications across multiple projects
- 🤖 Auto-fill job details using AI (Claude, ChatGPT, Gemini)
- 💼 Manage interviews and contacts
- 📊 Export data to Excel/CSV
- 🗺️ Get directions to interview locations
- 🎨 Beautiful light/dark themes
- And much more!
